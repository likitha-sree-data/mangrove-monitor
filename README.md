# Mangrove Monitor

Real-time global mangrove loss monitoring dashboard.

## Stack
- **Ingestion**: Python + Snowflake connector
- **Storage**: Snowflake (RAW / STAGING / MART schemas)
- **Transformation**: dbt
- **Dashboard**: Streamlit
- **Scheduling**: GitHub Actions

## Data Sources
- FAO World's Mangroves 2000-2020 (2023)
- Global Mangrove Alliance State of World's Mangroves 2024
- UC Santa Cruz / World Bank 2024
- IUCN Red List of Mangrove Ecosystems 2024
